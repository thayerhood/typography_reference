CSS Typography Reference

Typography Properties

Property Name Description
font-family Font Family Specifies the font for an element.
font-weight Font Weight Sets how thick or thin characters in text should be displayed.
font-style Font Style Mostly used to specify italic text.
text-transform Text Transform Controls the capitalization of text.
letter-spacing Letter Spacing Controls the spacing between text characters.
word-spacing Word Spacing Adjusts the white space between words, increasing or decreasing.
line-height Line Height Specifies the height of a line.
text-align Text Align Specifies the horizontal alignment of text in an element.
